Introduction
============

Varians product allows you to have buayable GetPaid products with following properties

* Physical, shipable, goods

* Multiple variants of the same good with different titles and prices, e.g. T-shirt sizes

* Multiple images for each product (not bound to variants)

Two new content types are created

* Multiple image product

	* Allows uploading multiple images of the product

* Variants product

	* Different buy portlet: one with pick variation option

Custom shopping portlet is provided for enhanced shopping experience
and to support product variation choices.

Installation
============

#. Include *gegpaid.atshop* Python egg in your buildout

#. Run add-on product installer

**WARNING**: Product ships with dependency to FancyZoom Javascript
library which is a non-free product.
`Read more about FancyZoom licensing terms <http://www.cabel.name/2008/02/fancyzoom-10.html>`_.
